## replaceVowels

2020/05/31

ให้เขียนฟังก์ชั่นรับ input เป็น string และ
return string ใหม่โดยเงื่อนไขคือ

- แทนที่ ตัวอักษร a, e, i, o, u ด้วยตัวเลข 1, 2, 3, 4, 5 ตามลำดับ

__ตัวอย่าง__

```
input 'hello'
return 'h2ll4'

input How are you today?
return H4w 1r2 y45 t4d1y?
```
